🗡️ Initiative Tracker
Консольное приложение для управления участниками боевой сцены DnD 5e, Pathfinder 2e и других. Позволяет добавлять существ, наносить урон/лечить, удалять, просматривать и сохранять состояние в файл.

📦 Возможности
➕ Добавление участника (имя, инициатива, КД, HP)
📋 Просмотр всех участников, отсортированных по инициативе (убывание)
❤️‍🩹 Изменение HP (положительное – лечение, отрицательное – урон)
🗑️ Удаление участника по индексу
🧹 Очистка всего поля боя
💾 Автосохранение в текстовый файл (combatants.txt)

🛠️ Установка и запуск
Убедитесь, что установлен Python 3.6 или выше.

Скачайте все файлы проекта:
main.py
models.py
storage.py
logic.py

Запустите приложение:
python main.py

🧩 Структура проекта
.
├── main.py          # Точка входа, главное меню
├── models.py        # Класс Creature (существо)
├── storage.py       # Чтение/запись в файл, глобальный список combatants
└── logic.py         # Логика работы: добавление, удаление, изменение HP, вывод

📖 Использование
После запуска отображается меню с пунктами:
1. Добавить
2. Показать
3. Урон/Хил
4. Удалить
5. Очистить поле
6. Выход

Выберите цифру и следуйте инструкциям.

Пример работы
Добавление
Введите имя, инициативу, КД и HP. Если ввести "-" на любом шаге – добавление отменяется.

Введите имя: Гоблин
Введите инициативу: 15
Введите КД: 12
Введите ХП: 10
Персонаж Гоблин добавлен!

Просмотр
Все участники выводятся с индексом, начиная с 0, отсортированные по инициативе.

Урон/Хил
Сначала показывается список, затем введите номер цели и число (отрицательное – урон, положительное – лечение). Если HP становится ≤ 0, существо автоматически удаляется.

Номер цели: 0
Число (положительное или отрицательное): -5
Гоблин получил урон в размере -5.
💀 Гоблин испустил дух.
Гоблин мёртв. Удаление из списка.

Удаление
Введите индекс участника для удаления.

Очистить поле
Удаляет всех участников и сохраняет пустой файл.

Выход – завершение программы.

💾 Сохранение
Данные автоматически сохраняются в файл combatants.txt при каждом изменении (добавление, удаление, изменение HP, очистка). Формат строки:

имя|инициатива|КД|HP
При запуске файл читается и восстанавливает состояние.

⚠️ Примечания
Код использует только стандартную библиотеку Python, дополнительные пакеты не требуются.

Все числовые поля проверяются на корректность; при ошибке ввода программа запросит повторный ввод.

Для выхода из ввода в некоторых пунктах используйте -.
